Welcome! This guide will walk you through installing VirtualBox and Kali Linux step by step. Use the links below to jump to any section.

Note: Virtualization must be enabled in your BIOS/UEFI.

Installing VirtualBox

  1. Go to the VirtualBox Downloads page.
  2. Download the installer for your OS (Windows, macOS, or Linux).
  3. Run the installer and follow the on-screen instructions (be sure to run as administrator).
  4. Accept default options unless you have specific needs.
  5. Once installed, launch VirtualBox to ensure it opens correctly.

Installing Kali Linux (for VirtualBox)

  1. Visit the Kali Linux Virtual Machines page.
  2. Download the Kali Linux VirtualBox Image (OVA file).
  3. Open the file.
  4. After opening the file, extract the VDI file (red color) to a suitable directory or a drive by right clicking on it and clicking extract all.
  5. Open VirtualBox, go to Add, and select the extracted VDI (.vdi) file from the directory or the drive.
  6. Kali Linux should naturally get set up inside VirtualBox
  7. Start the imported Kali Linux VM.

Note: The default login details are username: kali, password: kali

Extracting & Setting Up

  1. If your Kali download is in a compressed format (e.g., .zip), extract it using built-in tools or 7-Zip (recommended).
  2. Ensure you have at least 30 GB free space before extraction.
  3. After extraction, proceed with the VirtualBox 'Add' steps.

Suggestion: Consider using a dedicated partition for Kali Linux to avoid space issues. Keep at least a 100 GB.

Troubleshooting & Tips

  • Enable Virtualization: If you get errors about virtualization, enable it in your BIOS/UEFI.
  • Network Issues: Set network mode to Bridged Adapter or NAT in VM settings.
  • Performance: Allocate more RAM/CPU if your system allows.
  • Disk Space: Ensure you have enough disk space for the VM and Kali Linux updates.

For more help, visit the VirtualBox Manual or the Kali Linux Documentation.
